# Priority Contacts Manager

## Introduction
The Priority Contacts Manager is an application designed to help users manage their important contacts efficiently. It allows users to categorize and prioritize their contacts based on their needs.

## Features
- Categorize contacts into various groups.
- Set priorities for each contact.
- Easy search functionality to find contacts quickly.
- User-friendly interface.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/ANIRUDDHA-PATKI/priority-circle.git
   cd priority-circle
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the application**:
   ```bash
   npm start
   ```

## Usage Guide
- To add a contact, navigate to the "Add Contact" section and fill in the information.
- Adjust priority settings using the provided interface.
- Use the search bar to quickly locate contacts.

## Technology Stack
- **Frontend**: React.js
- **Backend**: Node.js with Express
- **Database**: MongoDB
- **Styling**: CSS
